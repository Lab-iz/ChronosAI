# Avisos de Terceiros

O CHRONOS-SEGURO foi desenhado para interoperar com as seguintes bibliotecas de terceiros:

- NumPy
- PyTorch
- PyTorch Geometric
- REBOUND
- FastAPI
- Pydantic
- Astropy
- Astroquery
- Matplotlib
- Pandas
- scikit-learn
- pytest

Cada dependencia continua sujeita a sua propria licenca.
